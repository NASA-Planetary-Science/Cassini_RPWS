RPWS Wideband Browse Dataset

The WBR browse images cover 1 hour periods and contain all data for a given band. Multiple dataset may have been used to create the spectrogram (all from the same hour and the same band setting). As the WBR has 2 band setting, this can result in up to 48 plots per day (assuming continuous coverage).
