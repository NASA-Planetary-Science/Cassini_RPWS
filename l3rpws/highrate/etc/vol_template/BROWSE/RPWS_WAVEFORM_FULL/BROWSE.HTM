RPWS Waveform Browse Dataset

The WFR browse images cover 24 hour periods and contain all data for a given band. Multiple dataset may have been used to create the spectrogram (all from the same day and the same band setting). As the WFR has 2 band setting, this can result in up to 2 plots per day.
