This package contains utilities often used when making PDS volumes.  There are
three modules:

  rpw_pds.index  - Indexer for volume files

  rpw_pds.util   - Day of Year class, version files, md5hashes, text
                   subtitution, logging setup and other misc. utilties.


  rpw_pds.parse  - Functions for PDS label parsing


  rpw_pds.verify - Wrappers for invoking pvt_tools and linklint.  The pvt_tools
                   can be found at:

                   	http://pds.jpl.nasa.gov/tools/software_download.cfm 

                   Select the link for the "PDS Tools Package".

Installation instrucitons can be found in INSTALL.txt.
